Centered Div

This div is centered both horizontally and vertically using 100vh!

Explanation: 100vh: height: 100vh; is used to make the body element take up the full height of the viewport (100% of the visible screen). Flexbox: display: flex; makes the body a flex container. justify-content: center; horizontally centers the child element (the div). align-items: center; vertically centers the child element. This approach ensures that the div remains centered regardless of viewport size, using 100vh to adapt to the height of the user's screen.

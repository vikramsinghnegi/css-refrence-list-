2. Relative Units Relative units change based on other factors like the parent element's size or the viewport size. Common relative units include: em: Relative to the font-size of the element (2em means 2 times the size of the current font). rem (root em): Relative to the font-size of the root element (). % (percent): Relative to the parent element. vw (viewport width): 1% of the viewport's width. vh (viewport height): 1% of the viewport's height. vmin: The smaller value of vw or vh. vmax: The larger value of vw or vh.

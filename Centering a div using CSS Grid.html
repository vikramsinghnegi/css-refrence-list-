Centered Div

This div is centered using CSS Grid!

Explanation: CSS Grid: display: grid; sets the body as a grid container. place-items: center; is a shorthand for centering both horizontally and vertically. It automatically applies align-items: center and justify-items: center. Why this method is easy: CSS Grid: It allows for centering with just one line: place-items: center;. No need to set extra properties like in Flexbox (justify-content and align-items), making it a quick and efficient method for centering.

Float Property Example

This image is floated to the **left** using the `float: left;` property. You can see how the text wraps around the image. The image takes up space on the left, while the text flows smoothly on the right.

This image is floated to the **right** using the `float: right;` property. Here, the image takes up space on the right, and the text is wrapped on the left.

This image is not floated; it simply appears in the normal document flow. The text starts below the image, as no floating is applied.
